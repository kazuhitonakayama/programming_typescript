## Overview
This repository is learing TypeScript with reading 『programming TypeScript』

## Script
### TSC（TypeScript Compile） and execute eslint

```
> yarn compile
```

### プログラムの実行

```
> yarn execute
```

## ESlintの実行

```
# lintの起動（Fixはしない）
> yarn lint:eslint

# lintの起動（Fixする）
> yarn fix:eslint
```
